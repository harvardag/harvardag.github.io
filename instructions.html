Instructions for AlumniMagnet Technical Team

This document contains all the necessary information to reskin the site according to the updated design. Each section contains instructions showing, in yellow, where PHP or other backend code needs to go. Ellipses in the code represent static content. When opening the HTML snippets (e.g. header.txt), a quick search for the text "ALUMNIMAGNET: " in the document will get you to the places highlighted on this page in yellow.

Links to files

All files and links in this skin point either to a full http address, or to a relative address relative to the root directory of the website. If the folders "assets" and "dist" are uploaded to the root directory, no recoding of links is necessary.

CSS

All CSS is either in the bootstrap core or in the file harvardag.css. The only thing that needs to be updated is the background image for the site. This can simply be uploaded with the same name as the current one and no further changes are needed.

Header

The file header.txt contains all the HTML preceding the contents of an article. Parts to be hard-coded include:

  • "Become a Member" or "My Account" button
  • "Sign In" or "Sign Out" button

Parts removed from the current design include:

  • "Hello, Member" message when logged in

Finally, the menu navigation follows the standard bootstrap format and is straightforward.

<!DOCTYPE html>
...
							<!-- BEGIN HELLO BLOCK -->
							<div class="hello">
								...
									<a href="Put link here" class="btn btn-success btn-header" role="button">
										Become a Member OR My Account
									</a>								
								...
									<a href="Put link here" class="btn btn-success btn-header" role="button">
										Sign In
									</a>
								...
							<!-- END HELLO BLOCK -->
						...
						<!-- BEGIN NAVBAR -->
						<ul class="nav navbar-nav navbar-right">
							...
								<a href="Put link here" class="dropdown-toggle" data-toggle="dropdown" role="button" aria-expanded="false">
									About
								</a>
								<ul class="dropdown-menu" role="menu">
									<li><a href="Put link here">Leadership Team</a></li>
									<li><a href="Put link here">Our Goals</a></li>
									<li><a href="Put link here">FAQ</a></li>
									<li><a href="Put link here">Contact Us</a></li>
								</ul>
							...
						<!-- END NAVBAR -->
						...
			<!-- END HEADER -->
			<!-- BEGIN MAIN PAGE CONTENTS -->
			<div class="container">
				<div class="row">
					<div class="col-xs-12 article">
					<!-- BEGIN ARTICLE IMAGE, TITLE, AND SUBHEAD -->
					...

Footer

The footer for the site is at footer.txt. There is only one dynamic piece to the footer, namely, the date in the copyright notice. This goes here:

...
								<strong>Legal</strong><br />
								Copyright © 2015 Harvard Alumni for Agriculture. All Rights Reserved.<br />
								<a href="http://www.harvardagriculture.org/article.html?aid=7" title="Privacy Policy and Terms of Service">Privacy and Terms</a>.
								...

Article Images, Title, and Subhead

The standard article image formatting works. On "article.html", the choices are Upper Right, In Text Upper Right, In Text Upper Left, or In Text Center. We don't use the event template. Simply paste the code (the one already on our website) into header.txt here:

...
						<!-- BEGIN ARTICLE IMAGE, TITLE, AND SUBHEAD -->
						...
						<!-- END ARTICLE IMAGE, TITLE, and SUBHEAD -->
						...

However, to make the formatting work, there are a couple "<br>" tags that must be removed. Note that these must be removed both on "article.html" as well as on the home page / portal page where "articles" are also shown.


	<!-- CURRENT CODE: -->
	<br><span class="article-top">ARTICLE TITLE</span><br><br>
	<span class="subhead">SUBHEAD TEXT</span><br><br>
	<!-- FIX #1: Remove first yellow "br" above -->
	<!-- FIX #2: Put second two yellow "br"'s above INSIDE the end of the subhead span -->
	
	<!-- UPDATED CODE: -->
	<span class="article-top">ARTICLE TITLE</span><br><br>
	<span class="subhead">SUBHEAD TEXT<br><br></span>

Article Contents

Make sure the "normaltext" element is a div not a paragraph

Home Page / Portal Page

The portal page has the same 3-column layout as all AlumniMagnet sites but it is done using Bootstrap rather than tables.
